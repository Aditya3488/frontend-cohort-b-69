Hello Welcome to the Frontend cohort Day 1 assignment

In this assignment, we will explore the use of essential HTML tags to create a functional and interactive webpage. We will begin by adding a welcoming heading using the h1 tag, followed by a paragraph using the p tag to briefly describe the purpose of the project. Anchor tags a will be used to create links that allow visitors to send emails, make phone calls, send WhatsApp messages, and navigate to external websites. We will also insert an image using the img tag and embed a video with playback controls through the video tag. Additionally, we will include an audio file using the audio tag to demonstrate its playback features. Finally, the assignment will showcase how to embed external content, such as a YouTube video using the iframe tag and an Instagram post using the blockquote and script tags. This project will help you understand the practical applications of these HTML elements.
